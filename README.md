🛡️ Ghost-Net: پروژه پیشرفته دور زدن فیلترینگ هوشمند (L7)
پروژه Ghost-Net یک راهکار جامع برای مقابله با فیلترینگ لایه ۷ و اینترنت ملی است. این پروژه با ترکیب پروتکل Reality، تکنولوژی XTLS-Vision و شبیه‌ساز رفتار ترافیک، اتصالی پایدار و غیرقابل شناسایی را فراهم می‌کند.
🚀 مراحل راهنمای نصب و راه‌اندازی (قدم‌به‌قدم)
۱. پیش‌نیازها
• یک سرور مجازی (VPS) با سیستم‌عامل Ubuntu 20.04 یا بالاتر.
• دسترسی به کاربر Root.
۲. اجرای اسکریپت نصب اصلی
ابتدا با استفاده از SSH به سرور خود متصل شوید و دستور زیر را برای شروع نصب خودکار وارد کنید:

sudo apt update && sudo apt install python3 curl -y && curl -Ls https://raw.githubusercontent.com/amirdavodi/Smart-GFW-Bypass/main/install.py | sudo python3


در طول نصب:
•اسکریپت از شما نام کاربری و رمز عبور برای پنل مدیریت (x-ui) می‌پرسد.
• از شما می‌خواهد یک پورت استراتژیک (مثل ۴۴۳ یا ۱۲۳) انتخاب کنید.
• در انتها، یک لینک VLESS اختصاصی و آدرس پنل را به شما نمایش می‌دهد. حتماً این اطلاعات را ذخیره کنید.
۳. تنظیم اینباند (Inbound) در پنل
پس از نصب، برای اینکه لینک اتصال کار کند، باید تنظیمات را در پنل ثبت کنید:
1. وارد پنل شوید (آدرس http://Server-IP:2053).
2. به بخش Inbounds رفته و روی Add Inbound کلیک کنید.
3. تنظیمات را دقیقاً طبق اطلاعاتی که اسکریپت در پایان نصب به شما داده پر کنید:
• Protocol: vless
• Port: همان پورتی که انتخاب کردید (مثلاً 443)
• Client UUID: همان آی‌دی تولید شده توسط اسکریپت
• Flow: حتماً روی xtls-rprx-vision تنظیم شود.
• Reality: تیک آن را بزنید و کلیدهای Private Key و Short ID که اسکریپت داده بود را وارد کنید.
• Dest & SNI: روی www.microsoft.com:443 تنظیم کنید.
۴. فعال‌سازی شبیه‌ساز رفتار (حیاتی)
برای اینکه ترافیک شما توسط سیستم‌های هوشمند شناسایی نشود، باید اسکریپت mimic.py را اجرا کنید تا ترافیک فیک تولید کند:

nohup python3 mimic.py &

استفاده از nohup باعث می‌شود حتی پس از بستن ترمینال، اسکریپت به کار خود ادامه دهد.
📱 نحوه اتصال و تست نهایی
1. لینک vless:// که اسکریپت به شما داده را کپی کنید.
2. در نرم‌افزار v2rayNG (اندروید) یا V2Box (آیفون)، گزینه Import from Clipboard را بزنید.
3. روی نام کانفیگ کلیک کنید و دکمه اتصال را بزنید.
4. تست سرعت: به سایت fast.com یا youtube.com بروید تا از پایداری و سرعت بالای اتصال خود مطمئن شوید.
🛠 ابزارهای موجود در این مخزن
• install.py: اسکریپت هوشمند برای نصب پنل، بهینه‌سازی هسته لینوکس (BBR) و تولید خودکار لینک اتصال.
• mimic.py: تولیدکننده ترافیک تصادفی برای فریب دادن سیستم‌های تحلیل رفتار (Traffic Analysis).
⚠️ نکات امنیتی و پایداری
• پورت‌های انتخابی: پورت‌های ۱۲۳ و ۳۴۷۸ معمولاً در لایه ۷ فیلترینگ اولویت کمتری برای مسدودسازی دارند.
• Fingerprint: در تنظیمات کلاینت، همیشه Fingerprint را روی chrome یا safari قرار دهید تا ترافیک شما دقیقاً شبیه مرورگر وب به نظر برسد.
• پاکسازی تاریخچه: اسکریپت به صورت خودکار تاریخچه دستورات سرور را پس از نصب پاک می‌کند تا امنیت شما حفظ شود.
